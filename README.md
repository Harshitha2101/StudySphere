# StudySphere

### Smart Student Dashboard UI

StudySphere is a modern and responsive student dashboard designed to simplify academic management through an elegant and interactive interface.

<br>

![StudySphere Screenshot](./github/screenshots/studysphere.png)

</div>

---

# Features

- Responsive Dashboard UI
- Dark / Light Theme Toggle
- Dynamic Timetable System
- Attendance Tracking
- Examination Schedule
- Announcements Section
- Teacher Leave Updates
- Mobile Responsive Design
- Modern Card-Based Layout
- Local Storage Theme Saving

- HTML5
- CSS3
- JavaScript
